NodeJS

Node.js is an open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside a web browser. Built on Chrome's V8 JavaScript engine, it enables developers to use JavaScript for server-side scripting, allowing for the development of both front-end and back-end web applications with a single language. Node.js is well-suited for building scalable and real-time applications due to its event-driven, non-blocking I/O model.
